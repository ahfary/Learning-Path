# Apa itu JavaScript?

Jika kita mengibaratkan website sebagai manusia:
* **HTML** adalah **Tulang & Kerangka** (Struktur).
* **CSS** adalah **Kulit & Baju** (Tampilan).
* **JavaScript** adalah **Otak & Otot** (Perilaku/Logika).

Tanpa JavaScript, website hanya diam saja (statis). Dengan JavaScript, website bisa berinteraksi, berpikir, dan merespons tindakan pengguna.

## Apa yang bisa dilakukan JS?
1.  **Manipulasi HTML/CSS:** Mengubah teks, warna, atau menyembunyikan elemen saat tombol diklik.
2.  **Validasi Form:** Mengecek apakah email yang dimasukkan valid sebelum dikirim.
3.  **Komunikasi Server:** Mengambil data dari database tanpa me-reload halaman (AJAX/Fetch).
4.  **Game & Animasi:** Membuat game interaktif di browser.
